# Pharmacy Management System

A desktop application built in Java for managing pharmacy inventory and operations, developed as a project for my Object-Oriented Programming I course.

## Overview

This project demonstrates core Object-Oriented Programming principles (encapsulation, inheritance, abstraction, and polymorphism) through a GUI-based desktop application that handles basic pharmacy inventory management via CRUD operations.

## Features

- **Add Medicine** – Add new medicine records to inventory with details (name, quantity, price, etc.)
- **View Records** – Display all medicine inventory records in a user-friendly interface
- **Update Records** – Edit existing medicine details (price, stock quantity, etc.)
- **Delete Records** – Remove medicine entries from inventory
- **GUI Interface** – Intuitive graphical interface built with Java Swing/AWT for ease of use

## OOP Principles Applied

- **Encapsulation** – Medicine and inventory data encapsulated within dedicated classes
- **Inheritance** – Class hierarchies used to organize shared behavior across components
- **Abstraction** – Core operations abstracted into reusable methods/classes
- **Polymorphism** – Method overriding/overloading used where applicable

## Tech Stack

- **Language:** Java
- **UI:** Java Swing / AWT (GUI)
- **Paradigm:** Object-Oriented Programming

## How to Run

1. Clone the repository:
   ```
   git clone https://github.com/SudiptaChowdhuryTirtha/Pharmacy-management-System.git
   ```
2. Open the project in your preferred Java IDE (e.g., IntelliJ IDEA, Eclipse, NetBeans)
3. Locate the main class inside the `PharmacyManagementSystem` folder
4. Run the main class to launch the GUI application
